# Background 관련 속성



### background-color

- hex, rgb, rgba



### background-image

- url()



### background-repeat

- repeat(default) | no-repeat



### background-size

- contain | cover | custom
- contain - 요소 안에 다 들어가면서 최대 사이즈로 조정
- cover - 빈 공간이 없도록 다 채울 수 있도록 사이즈 조정. 잘리는 부분이 있을 수 있음



### background-position

- x, y 위치를 명시해야 함
- center, left, top, px, % 등
