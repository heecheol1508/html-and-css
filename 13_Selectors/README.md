# Selectors





### Type, Class & ID Selector



### Child, Descendant & Sibling Combinators

- Child Combinator
  - `parent > child`
- Descendant Combinator
  - `parent descendant`
- Sibling Combinators
  - `selected + sibling`
  - `selected ~ sibling`



### Structural Pseudo-classes

- `element:first-child`
- `element:last-child`
- `element:nth-child(n)`



### User Action Pseudo-classes

- `element:hover`
- `element:active`
- `element:focus`
