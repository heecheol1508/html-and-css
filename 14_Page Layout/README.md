## Bootstrap

반응형 Grid System을 쉽게 CSS로 구현할 수 있게 도와주는 CSS Framework

```html
<div class="container">
    <div class="row">
        <div class="col-12 col-md-6">
            <!-- 반드시 이 구조를 지키면 grid 바로 적용됨 -->
        </div>
        <div class="col-12 col-md-6">
            <!-- 반드시 이 구조를 지키면 grid 바로 적용됨 -->
        </div>
    </div>
</div>
```
