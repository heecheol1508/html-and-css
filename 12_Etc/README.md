### Box Shadow

```css
box-shadow: h-offset v-offset blur spread color;
```



### Opacity

```css
opacity: 0; /* 0: 투명, 1: 불투명 */
```



### Overflow

```css
overflow: visible; /* visible | auto | scroll | hidden */
```
